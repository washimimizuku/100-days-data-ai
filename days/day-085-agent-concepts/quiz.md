# Day 85: Agent Concepts - Quiz

Test your understanding of AI agent concepts.

---

### Question 1
What is the primary difference between an AI agent and a single LLM call?

A) Agents are faster than LLM calls  
B) Agents can iterate, use tools, and self-correct  
C) Agents don't use language models  
D) Agents are always more accurate  

**Correct Answer: B**

---

### Question 2
Which component of an agent is responsible for maintaining context across interactions?

A) Perception  
B) Reasoning  
C) Action  
D) Memory  

**Correct Answer: D**

---

### Question 3
In the agent loop pattern, what happens after an action is executed?

A) The agent terminates  
B) Memory is updated with the result  
C) All tools are reset  
D) The goal is changed  

**Correct Answer: B**

---

### Question 4
What type of agent learns from experience and adapts its strategies over time?

A) Simple Reflex Agent  
B) Goal-Based Agent  
C) Utility-Based Agent  
D) Learning Agent  

**Correct Answer: D**

---

### Question 5
In the ReAct pattern, what comes after "Thought"?

A) Observation  
B) Action  
C) Memory  
D) Goal  

**Correct Answer: B**

---

### Question 6
Which agent design principle refers to operating independently with minimal human intervention?

A) Reactivity  
B) Proactivity  
C) Autonomy  
D) Social Ability  

**Correct Answer: C**

---

### Question 7
What is a key challenge when deploying AI agents in production?

A) Agents are too simple  
B) Managing reliability, safety, and cost  
C) Agents don't use LLMs  
D) Agents can't use tools  

**Correct Answer: B**

---

### Question 8
Which memory type stores persistent facts and knowledge?

A) Short-term memory  
B) Working memory  
C) Long-term memory  
D) Episodic memory  

**Correct Answer: C**

---

### Question 9
What is the purpose of the "Reflection" pattern in agents?

A) To speed up execution  
B) To evaluate results and improve approach  
C) To delete memory  
D) To disable tools  

**Correct Answer: B**

---

### Question 10
Which capability allows agents to call external functions like search engines or calculators?

A) Memory  
B) Planning  
C) Tool Use  
D) Reasoning  

**Correct Answer: C**

---

## Answer Key
1. B - Agents can iterate, use tools, and self-correct
2. D - Memory
3. B - Memory is updated with the result
4. D - Learning Agent
5. B - Action
6. C - Autonomy
7. B - Managing reliability, safety, and cost
8. C - Long-term memory
9. B - To evaluate results and improve approach
10. C - Tool Use
