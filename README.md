# 100 Days of Data and AI

A structured 100-day program (1 hour/day) to build foundational skills in Data Engineering, Machine Learning, and Generative AI.

## Program Structure

### Phase 1: Data Engineering Foundations (Days 1-35)
Build core data engineering skills with modern tools and formats.

### Phase 2: Development & Quality (Days 36-50)
Learn APIs, testing, and data quality practices.

### Phase 3: Machine Learning Foundations (Days 51-70)
Master ML fundamentals and MLOps basics.

### Phase 4: GenAI & LLMs (Days 71-92)
Explore LLMs, RAG systems, and agentic AI.

### Phase 5: Integration & Practice (Days 93-100)
Combine concepts and prepare for projects.

## What You'll Learn

### Data Engineering
- Data formats (CSV, JSON, Parquet, Avro, Arrow)
- Table formats (Apache Iceberg, Delta Lake)
- Apache Spark (DataFrames, SQL, Streaming)
- Apache Kafka (Real-time streaming, producers, consumers, Kafka Streams)
- Data quality (Great Expectations, profiling, validation)
- dbt & Airflow (Integration in mini projects - assumes prior knowledge)
- Data architecture (Medallion, Data Mesh)

### Development Skills
- FastAPI (REST APIs, async, authentication)
- Docker (Containerization basics)
- Git (Version control workflows)
- Testing (pytest fundamentals)
- PostgreSQL (Database basics)

### Cloud Basics
- AWS S3 (Object storage)
- AWS EC2 (Compute basics)
- AWS Lambda (Serverless)

### Machine Learning
- Scikit-learn (Classical ML)
- PyTorch (Deep learning basics)
- Hugging Face (Transformers)
- MLflow (Experiment tracking)
- Model evaluation and monitoring

### GenAI & LLMs
- LLM fundamentals
- Prompt engineering
- RAG systems (Retrieval-Augmented Generation)
- Vector databases (ChromaDB, FAISS)
- LangChain & LangGraph
- Agentic AI basics
- Local LLMs (Ollama)

### Specialized Topics
- Computer vision basics
- NLP fundamentals
- Audio AI introduction
- Reinforcement learning basics

## Weekly Breakdown

### Week 1-2: Data Formats & Processing
Learn modern data formats and when to use them.

### Week 3-4: Table Formats & Architecture
Master Iceberg, Delta Lake, and data architecture patterns.

### Week 5: Apache Spark
Build distributed data processing skills.

### Week 6-7: Streaming & Data Quality
Learn Kafka for real-time streaming and data quality frameworks. Airflow integration shown in mini projects (assumes prior knowledge).

### Week 8: Data Quality & Development
Implement data quality checks and learn essential dev tools.

### Week 9: APIs & Databases
Build REST APIs with FastAPI and work with PostgreSQL.

### Week 10-11: Machine Learning
Master ML fundamentals and MLOps practices.

### Week 12-13: GenAI & LLMs
Explore LLMs, RAG, and agentic AI.

### Week 14: Integration & Specialization
Combine concepts and explore specialized AI topics.

### Week 15: Final Projects
Apply everything learned in integrated projects.
